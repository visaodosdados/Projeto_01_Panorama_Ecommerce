# Panorama do E-commerce Brasileiro

Projeto desenvolvido por Amanda Soler.

## Objetivo

Analisar dados de um marketplace brasileiro para compreender padrões de vendas, comportamento dos clientes e desempenho dos produtos.

## Tecnologias Utilizadas

- Python
- Pandas
- Numpy
- Google Colab
- Power BI
- GitHub
